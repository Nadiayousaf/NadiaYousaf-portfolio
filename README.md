## Setup / Installation
1. Clone the repository:

```bash
git clone https://github.com/NadiaYousaf/Nadia_Yousaf_Portfolio_FullStack.git
````

2. Navigate to the backend folder:

```bash
cd backend
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. **Important:** Add your Google service account JSON key in the `backend` folder (do **not** push it to GitHub). Update `app.py` with the correct path.

5. Run the backend:

```bash
python app.py
```
6. Open `index.html` in your browser to view the portfolio.
